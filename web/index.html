NASA POWER 小时气温拉取工具

输入经纬度，浏览器直接拉取 NASA POWER Hourly T2M，计算超温占比并导出 Excel。

按城市名添加点位

输入城市名后点击查询，先查内置城市表；未命中时查询 OpenStreetMap Nominatim，确认后加入下方 CSV。

不会自动联想；只有点击查询时才访问公共地理编码服务。

地名查询 fallback 使用 OpenStreetMap Nominatim。© OpenStreetMap contributors。公共地理编码服务不适合批量自动解析。

按国家列出城市

输入国家名称后点击查询，列出内置 Top 经济城市候选，再手动加入下方 CSV。

不会自动批量加入；候选为内置实用表，不是官方 GDP 排名。

国家城市列表来自工具内置实用候选；未内置国家请用城市名单点查询或手工经纬度输入。

内嵌说明

数据口径

数据源为 NASA POWER Hourly API，参数为 T2M，即 2 米气温小时平均值，单位摄氏度。默认使用 LST 当地太阳时。

隐私边界

坐标、缓存和导出结果都在浏览器本地处理。本工具没有自有服务器；只有 NASA API 会收到经纬度请求。

缓存说明

缓存保存在当前浏览器的 IndexedDB 中。换电脑、换浏览器或清空站点数据后，缓存不会自动保留。

本地运行文件

复制整个 web 文件夹到本地即可运行，至少需要 index.html、app.js、styles.css 三个文件保持在同一目录。
